# Local Server (Only for local usage)

Запуск подвійним кліком:
- `Start Local Server.command`

Що робить:
- запускає локальний `backend` (`http://localhost:5050`)
- запускає локальний `frontend` (`http://localhost:5173`)
- автоматично чистить "завислі" процеси на портах `5050` і `5173`

Якщо файл `.command` відкривається як текст:
- натисніть правою кнопкою на `Start Local Server.command` -> `Open`
- або запускайте `Start Local Server.app`
